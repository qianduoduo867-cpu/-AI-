# Remote AI Control Agent

远程AI运维控制系统 - 可以在任意设备通过AI远程执行命令

## 架构

        ┌──────────────┐         ┌──────────────┐
        │   服务端      │◄───────│   客户端    │
        │  (AI控制)    │ Socket │  (目标设备) │
        └──────────────┘         └──────────────┘

## 使用方法

### 1. 安装依赖

```bash
pip install openai
```

### 2. 启动客户端（目标设备）

```bash
python client.py <服务端IP> <端口>
# 例: python client.py 192.168.1.100 5555
```

### 3. 启动服务端（控制端）

```bash
python server.py 5555
```

### 4. 使用AI执行命令

在服务端交互界面输入:

```
ai 帮我安装nginx
```

AI会自动转换为Linux命令并发送给客户端执行。

## 命令说明

- `clients` - 查看已连接的客户端
- `exec <命令>` - 直接执行命令
- `ai <自然语言>` - AI理解后执行
- `quit` - 退出

## 安全说明

- 仅在内网使用，不要暴露到公网
- 生产环境需要加入认证和加密
- 建议配合防火墙限制访问
